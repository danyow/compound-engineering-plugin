# 关键模式模板

在向 `docs/solutions/patterns/cora-critical-patterns.md` 添加模式时使用此模板：

---

## N. [模式名称]（始终必需）

### ❌ 错误写法（[将导致 X 错误]）
```[language]
[展示错误方法的代码]
```

### ✅ 正确写法
```[language]
[展示正确方法的代码]
```

**原因：**[技术解释为什么这是必需的]

**适用位置/上下文：**[此模式适用的场景]

**相关文档：**`docs/solutions/[category]/[filename].md`

---

**使用说明：**
1. 将 N 替换为下一个模式编号
2. 将 [模式名称] 替换为描述性标题
3. 在"错误写法"中填入导致问题的代码
4. 在"正确写法"中填入解决方案
5. 在"原因"中解释技术原因
6. 在"适用位置/上下文"中说明此模式的适用场景
7. 链接到最初解决此问题的完整故障排查文档
